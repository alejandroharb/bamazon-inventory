# Bamazon Inventory App

**Back-End Technologies:** Node.js, Express

**Database:** MySQL

## Three Features
1. Customer Feature
    * Purchase Items
    * Inventory Automatically Updates
2. Manager Feature
    * View Products for Sale
    * Vew Low Inventory (less than 5 in stock)
    * Add to Inventory
    * Add New Product
3. Supervisor Feature
    * See Department Data
    * Create New Department
